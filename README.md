# Employee Management System

A full-stack Employee Management System built with React, Spring Boot, and MySQL.

## Features

- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Form Validation
- REST API

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Bootstrap

### Backend
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

### Backend

```bash
mvn spring-boot:run
```

### Frontend

```bash
npm install
npm run dev
```
